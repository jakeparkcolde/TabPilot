# TabPilot Chrome 웹 스토어 소개

## 한 줄 소개

작업 중인 탭은 보호하고, 잊고 있던 탭만 조용히 정리하는 Chrome 메모리 매니저

## 짧은 설명

오래 사용하지 않은 탭을 닫지 않고 안전하게 휴면 처리해 Chrome 메모리 사용을 줄입니다.

## 상세 설명

탭은 계속 늘어나지만, 모두 닫아버릴 수는 없습니다.

TabPilot은 현재 작업 중인 탭과 중요한 사이트를 보호하면서 오래 사용하지 않은 탭만 Chrome의 네이티브 휴면 기능으로 정리합니다. 탭과 URL은 그대로 유지되며, 다시 클릭하면 페이지가 자동으로 로드됩니다.

### 주요 기능

- 최초 실행 안내와 보호 사이트 프리셋
- 오래 사용하지 않은 탭을 정리 후보로 자동 분류
- 탭을 닫지 않고 휴면 처리해 메모리 확보
- 활성 탭, 고정 탭, 오디오 재생 탭, 로딩 중인 탭 자동 제외
- Google Docs, Figma, Notion 등 중요 도메인 기본 보호
- 사용자 지정 보호 도메인과 개별 탭 보호
- 원클릭 안전 정리와 선택 정리
- 시스템 여유 메모리가 설정 기준 이하일 때 실행되는 선택형 자동 정리
- 방금 정리한 탭 일괄 되돌리기
- 휴면 탭 목록과 빠른 재활성화
- 전체 탭 검색·필터·보호·휴면 관리
- 휴면 탭과 전체 정리 기록 통합 관리
- 휴면 탭 개수 배지와 메모리 절약 추정치
- 요약형, 선택형, 가이드형 팝업 레이아웃
- 시스템, 라이트, 다크 화면 테마

### 안전한 정리

TabPilot은 정리 직전에 탭 상태를 다시 확인합니다. 현재 사용 중이거나 보호 조건에 해당하는 탭은 휴면 처리하지 않습니다.

자동 정리는 기본적으로 꺼져 있습니다. 사용자가 설정에서 직접 활성화한 경우에만 동작합니다.

### 개인정보 보호

- 탭 제목과 도메인을 외부 서버로 전송하지 않습니다.
- 광고, 추적 도구, 사용자 계정을 사용하지 않습니다.
- 설정과 정리 기록은 현재 기기의 Chrome 저장소에만 보관합니다.
- 모든 사이트의 콘텐츠를 읽는 권한을 요청하지 않습니다.

TabPilot은 탭을 무작정 닫는 도구가 아닙니다. 작업 흐름은 유지하면서 Chrome이 사용하는 메모리를 더 쉽게 관리할 수 있도록 돕습니다.

## 권한 사용 목적

- `tabs`: 탭 상태 확인, 정리 후보 판단, 휴면 처리
- `storage`: 설정, 보호 목록, 정리 기록 저장
- `system.memory`: 시스템 가용 메모리와 절약 추정치 표시
- `alarms`: 사용자가 활성화한 자동 정리 주기 실행
