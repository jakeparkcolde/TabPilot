# TabPilot 개인정보처리방침

최종 업데이트: 2026년 6월 25일

TabPilot은 오래 사용하지 않은 Chrome 탭을 안전하게 휴면 처리하기 위한 브라우저 확장 프로그램입니다.

## 처리하는 정보

TabPilot은 기능 제공을 위해 다음 정보를 기기 내에서 처리합니다.

- 열린 탭의 제목, URL, 도메인
- 탭의 활성, 고정, 오디오 재생, 로딩, 휴면 상태
- 탭의 마지막 접근 시각
- 사용자가 설정한 보호 도메인과 개별 탭 보호 상태
- 탭 정리 시각, 결과, 재활성화 시각
- 시스템 전체 및 사용 가능한 메모리 용량

웹 탐색 활동에 해당할 수 있는 탭 URL과 도메인은 정리 후보를 판정하고 사용자가 목록을 확인하는 기능에만 사용됩니다.

## 저장과 보관

- 설정, 보호 도메인과 최근 정리 기록은 `chrome.storage.local`에 저장됩니다.
- 개별 탭 보호와 일부 실행 상태는 `chrome.storage.session`에 저장됩니다.
- 최근 정리 기록은 최대 100건을 보관합니다.
- 사용자는 설정 화면에서 정리 기록을 삭제할 수 있습니다.
- 확장 프로그램을 제거하면 Chrome이 확장 프로그램 저장 데이터를 제거합니다.

## 외부 전송과 공유

TabPilot은 처리하는 정보를 개발자 또는 제3자 서버로 전송하지 않습니다.

- 사용자 계정이나 로그인 기능을 제공하지 않습니다.
- 광고, 분석, 추적 또는 사용자 프로파일링 도구를 사용하지 않습니다.
- 사용자 정보를 판매하거나 제3자에게 공유하지 않습니다.
- 원격 코드를 다운로드하거나 실행하지 않습니다.
- 사람이 사용자의 탭 정보나 정리 기록을 열람할 수 있는 기능을 제공하지 않습니다.

## 권한 사용

- `tabs`: 탭 상태 확인, 정리 후보 판정, 휴면 처리와 재활성화
- `storage`: 설정, 보호 목록과 정리 기록의 로컬 저장
- `system.memory`: 시스템 메모리 참고 정보 표시
- `alarms`: 사용자가 활성화한 자동 정리 주기 실행

TabPilot은 모든 웹사이트의 콘텐츠를 읽는 호스트 권한이나 콘텐츠 스크립트를 사용하지 않습니다.

## Chrome 웹 스토어 제한적 사용 준수

TabPilot이 Chrome API를 통해 처리하는 정보는 사용자에게 명시된 단일 목적, 즉 안전한 탭 휴면 및 메모리 관리 기능을 제공하는 데만 사용됩니다. 해당 정보를 광고, 신용 평가, 판매, 사용자 프로파일링 또는 기능과 무관한 목적으로 사용하거나 전송하지 않습니다.

TabPilot의 정보 사용은 Chrome 웹 스토어 사용자 데이터 정책과 제한적 사용 요건을 준수합니다.

## 변경

개인정보 처리 방식이 변경되면 이 문서의 최종 업데이트 날짜와 내용을 수정합니다. 새로운 외부 전송 또는 새로운 정보 사용이 필요한 경우 사용자에게 명확하게 알리고 필요한 동의를 받습니다.

## 문의

개인정보 및 지원 문의는 Chrome 웹 스토어 목록에 표시된 게시자 지원 연락처를 이용해 주세요.
