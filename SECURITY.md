# Security Policy

## Supported version

Security fixes are applied to the latest published TabPilot version.

## Reporting a vulnerability

Do not include sensitive tab data, URLs, credentials, cookies or personal information in a public issue.

Report a vulnerability through the publisher support contact shown on the Chrome Web Store listing. Include:

- affected TabPilot and Chrome versions
- reproduction steps
- expected and actual behavior
- security impact

You should receive an initial response within seven days. Please allow time to investigate and publish a coordinated fix before public disclosure.
