# Chrome 웹 스토어 제출 입력값

## 기본 정보

- 이름: `TabPilot`
- 카테고리: `생산성`
- 언어: `한국어`
- 공개 범위 권장: 첫 심사는 `비공개 테스트` 또는 `링크로만 공개`, 검증 후 `공개`
- 단일 목적:
  - `사용자가 중요 탭을 보호하면서 오래 사용하지 않은 Chrome 탭을 안전하게 휴면 처리해 메모리를 관리하도록 돕습니다.`

## 짧은 설명

작업 중인 탭은 보호하고, 오래 사용하지 않은 탭만 안전하게 휴면 처리해 Chrome 메모리를 관리합니다.

상세 설명은 [STORE_LISTING_KO.md](STORE_LISTING_KO.md)를 사용합니다.

## 권한 사유

### tabs

열린 탭의 상태, 마지막 접근 시각, 제목과 URL을 확인해 안전한 정리 후보를 판정하고, 사용자가 실행한 탭 휴면 및 재활성화 기능을 제공합니다.

### storage

비활성 기준, 저메모리 자동 정리 설정, 화면 테마, 팝업 레이아웃, 보호 도메인과 최근 정리 기록을 사용자의 Chrome 로컬 저장소에 보관합니다.

### system.memory

시스템 전체 및 사용 가능한 메모리를 팝업에 참고 정보로 표시하고, 정리 전후의 시스템 가용 메모리 차이를 이용해 절약량 추정치를 제공합니다.

### alarms

자동 정리를 사용자가 명시적으로 활성화한 경우에만 설정된 주기로 시스템 여유 메모리를 확인하며, 사용자가 선택한 기준 이하일 때 안전한 정리 후보를 휴면 처리합니다. 자동 정리는 기본적으로 꺼져 있습니다.

## 원격 코드

- 답변: `아니요, 원격 코드를 사용하지 않습니다.`
- 설명: 모든 JavaScript와 CSS는 확장 프로그램 ZIP에 포함되며 외부 코드, `eval`, CDN 스크립트 또는 원격 모듈을 실행하지 않습니다.

## 데이터 공개

TabPilot은 기능 제공을 위해 웹 탐색 활동에 해당할 수 있는 탭 제목, URL과 도메인을 기기 내에서 처리합니다.

- 외부 서버 전송: 없음
- 판매 또는 제3자 공유: 없음
- 광고 또는 프로파일링: 없음
- 사람의 데이터 열람: 없음
- 저장 위치: `chrome.storage.local`, `chrome.storage.session`
- 보관: 정리 기록 최대 100건, 사용자가 삭제 가능

대시보드의 데이터 유형 선택지는 실제 표시 문구를 확인해 `웹 기록/웹 탐색 활동`에 해당하는 항목을 선택합니다. 개인정보처리방침과 대시보드 신고 내용은 반드시 동일하게 유지합니다.

## 제한적 사용 인증

모든 항목을 실제 동작과 대조한 후 인증합니다.

- 단일 목적과 직접 관련된 기능에만 사용
- 허용되지 않은 제3자 전송 없음
- 개인화 광고에 사용하지 않음
- 사람의 데이터 열람을 허용하지 않음

## 심사자 테스트 안내

1. 확장 아이콘을 클릭해 전체 탭, 후보 탭과 시스템 메모리를 확인합니다.
2. 설정에서 비활성 기준을 15분으로 선택합니다.
3. 15분 이상 사용하지 않은 일반 HTTP/HTTPS 탭을 준비합니다.
4. 팝업에서 후보 탭의 `휴면` 또는 `안전 정리하기`를 실행합니다.
5. 탭이 닫히지 않고 휴면 목록에 표시되는지 확인합니다.
6. 휴면 목록의 `열기`를 눌러 탭이 다시 로드되는지 확인합니다.
7. 활성, 고정, 오디오 재생, 로딩 또는 보호 도메인 탭은 후보에서 제외됩니다.
8. 자동 정리는 설정에서 사용자가 켜기 전까지 실행되지 않습니다.

로그인이나 테스트 계정은 필요하지 않습니다.
