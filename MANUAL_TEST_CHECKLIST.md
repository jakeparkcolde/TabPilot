# TabPilot Chrome 수동 검증 체크리스트

> 대상 빌드: `dist/`  
> 권장 Chrome: 121 이상  
> 목적: 자동 테스트로 확인할 수 없는 실제 Chrome API와 사용자 흐름 검증

## 1. 설치

- `npm run build`를 실행한다.
- Chrome에서 `chrome://extensions`를 연다.
- 개발자 모드를 활성화한다.
- `압축해제된 확장 프로그램을 로드합니다`에서 `dist/`를 선택한다.
- TabPilot 카드에 오류가 표시되지 않는지 확인한다.
- 서비스 워커 검사 화면에 초기 오류가 없는지 확인한다.

## 2. 대시보드

- 확장 프로그램 아이콘을 눌렀을 때 팝업이 열린다.
- 전체 탭 수가 현재 프로필의 열린 탭 수와 일치한다.
- 휴면 탭 수와 정리 후보 수가 숫자로 표시된다.
- 사용 가능 시스템 메모리가 표시되거나 `확인 불가`로 안전하게 대체된다.
- 새로고침 버튼이 최신 상태를 다시 불러온다.
- 설정 버튼이 별도 설정 페이지를 연다.

## 3. 안전 제외 조건

각 조건의 탭이 후보 목록에서 제외되는지 확인한다.

- 현재 활성 탭
- 고정 탭
- 오디오 재생 중인 탭
- 로딩 중인 탭
- 이미 휴면 상태인 탭
- `localhost` 또는 `127.0.0.1`
- `docs.google.com`, `figma.com`, `notion.so`
- 세션 보호 버튼을 누른 탭
- `chrome://settings` 같은 Chrome 내부 페이지

## 4. 설정

- 비활성 기준을 변경하고 저장한다.
- 최대 정리 탭 수를 변경하고 저장한다.
- 팝업을 다시 열었을 때 새 설정이 적용된다.
- `github.com`을 보호 도메인으로 추가하면 GitHub 탭이 후보에서 제외된다.
- `github.com/path` 입력은 거부된다.
- 같은 도메인의 중복 추가가 거부된다.
- 보호 도메인을 삭제하면 조건을 만족하는 탭이 다시 후보가 된다.
- Chrome을 재시작해도 설정과 보호 도메인이 유지된다.
- 기본값 복원이 60분, 최대 5개, 기본 보호 목록으로 되돌린다.

## 5. 개별 휴면

- 후보 탭의 `휴면` 버튼을 누른다.
- 해당 탭은 닫히지 않고 탭 스트립에 남아 있다.
- Chrome 탭 상태가 휴면으로 변경된다.
- 현재 탭으로 이동하면 페이지가 다시 로드된다.
- 설정 페이지 기록에 휴면 시간과 다시 사용한 시간이 표시된다.

## 6. 일괄 안전 정리

- 최대 정리 개수를 2개로 설정한다.
- 후보 탭을 3개 이상 준비한다.
- `안전 정리하기` 실행 시 최대 2개만 휴면 처리된다.
- 성공, 건너뜀, 실패 개수가 팝업에 표시된다.
- 실행 직전에 후보 탭 하나를 활성화하면 해당 탭은 건너뛰어진다.
- 탭 하나가 실행 중 닫혀도 나머지 후보는 계속 처리된다.

## 7. 세션 보호

- 후보 탭의 `보호` 버튼을 누르면 후보에서 즉시 사라진다.
- 팝업의 보호 탭 수가 증가한다.
- 팝업을 닫았다 다시 열어도 보호가 유지된다.
- Chrome을 완전히 종료하고 다시 열면 개별 탭 보호는 초기화된다.
- 영구 보호 도메인은 초기화되지 않는다.

## 8. 정리 기록

- 일괄 정리와 개별 정리 기록이 모두 표시된다.
- 성공과 실패 상태가 구분된다.
- 최신 기록이 위에 표시된다.
- 기록 삭제 후 목록이 비어 있다.
- 기록 삭제 후 기존 탭을 다시 활성화해도 삭제된 기록이 복원되지 않는다.

## 9. 다중 창 및 안정성

- Chrome 창 두 개 이상에서 후보가 집계된다.
- 다른 창의 오래된 탭이 정상적으로 후보가 된다.
- 탭 20개, 50개, 가능하면 100개 환경에서 팝업이 멈추지 않는다.
- 팝업을 빠르게 여러 번 열고 닫아도 오류가 발생하지 않는다.
- 정리 버튼을 연속 클릭해도 중복 정리가 실행되지 않는다.

## 10. 통과 기준

- 보호 조건에 해당하는 탭이 한 번도 잘못 휴면 처리되지 않는다.
- 휴면 처리로 탭 자체가 닫히지 않는다.
- 서비스 워커에 처리되지 않은 오류가 없다.
- 설정과 기록이 의도한 저장 수명에 맞게 유지된다.
- 모든 실패 상황에서 다른 탭 정리가 계속된다.
