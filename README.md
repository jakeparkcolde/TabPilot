# TabPilot

작업 중인 탭은 보호하고, 오래 사용하지 않은 탭만 안전하게 휴면 처리하는 Chrome 메모리 관리 확장 프로그램입니다.

TabPilot은 탭을 닫지 않습니다. Chrome의 네이티브 탭 휴면 기능을 사용하므로 탭과 URL은 유지되며, 다시 열면 페이지가 새로 로드됩니다.

## 주요 기능

- 최초 실행 온보딩과 보호 사이트 프리셋
- 전체 탭, 정리 후보, 휴면 탭 현황
- 활성·고정·오디오 재생·로딩 탭 자동 제외
- 보호 도메인과 브라우저 세션 단위 개별 탭 보호
- 수동, 선택, 자동 정리
- 시스템 여유 메모리 기준에 따른 조건부 자동 정리
- 시스템·라이트·다크 테마
- 방금 정리한 탭 일괄 되돌리기
- 휴면 탭 목록과 빠른 재활성화
- 전체 탭 검색·필터·보호·휴면 관리 화면
- 휴면 탭과 전체 정리 기록 통합 관리
- 시스템 전체·여유 메모리와 정리 가능량 추정
- A/B/C 팝업 레이아웃
- 최근 100건 정리 기록
- 외부 서버, 광고, 분석 도구 없음

## 권한

| 권한            | 사용 목적                                                            |
| --------------- | -------------------------------------------------------------------- |
| `tabs`          | 탭 상태와 마지막 접근 시각 확인, 보호 조건 판정, 탭 휴면 및 재활성화 |
| `storage`       | 설정, 보호 도메인, 정리 기록을 Chrome 로컬 저장소에 저장             |
| `system.memory` | 시스템 전체 및 사용 가능한 메모리 표시                               |
| `alarms`        | 사용자가 직접 활성화한 자동 정리 주기 실행                           |

호스트 권한, 콘텐츠 스크립트, `all_urls` 권한은 사용하지 않습니다.

## 로컬 설치

```bash
npm ci
npm run build
```

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. `압축해제된 확장 프로그램을 로드`를 누릅니다.
4. 이 프로젝트의 `dist` 폴더를 선택합니다.

## 개발

```bash
npm run typecheck
npm test
npm run lint
npm run build
```

출시 ZIP 생성:

```bash
npm run release
```

결과물은 `release/tabpilot-<version>-chrome-web-store.zip`에 생성됩니다.

## 개인정보

탭 제목, URL, 도메인과 정리 기록은 기능 제공을 위해 기기 내에서만 처리됩니다. 개발자 서버로 전송하거나 광고·프로파일링에 사용하지 않습니다.
